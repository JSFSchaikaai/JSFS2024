Есть сущность "пользователь" (таблица "users"). У "пользователя" есть "фамилия", "имя" и уникальный "id пользователя".

Есть "отношения пользователей" (таблица "user_relations"). "Отношение" содержит ссылку на пользователя ("id пользователя"),
ссылку на другого пользователя ("id пользователя" из той-же таблицы "users") и тип отношений (enum: "друг", "враг", "сосед", "родственник").
Таблица "user_relations" служит для связывания других таблиц (связывания одной и той-же таблици самой на себя) и не имеет собственного первичного ключа.

Есть сущность "группа" (таблица "groups"). У группы есть "название", "описание" и уникальный "id группы".

Есть "членство в группе" (таблица "group_members"). "Членство в группе" содержит пары "id пользователя" и "id группы".
Таблица "group_members" служит для связывания других таблиц и не имеет собственного первичного ключа.
