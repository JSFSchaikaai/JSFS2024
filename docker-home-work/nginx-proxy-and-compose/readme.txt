Есть web-сервер на node.js. Выполнен в виде образа my-web-server.
Работает на порту 3000. Изготовлен на основе образа node.

Есть прокси-сервер на основе nginx. Выполнен в виде образа my-proxy-nginx.
Принимает запросы на порту 8000 и перенаправляет их к web-серверу описанному выше.
Изготовлен на основе образа nginx.

И есть docker-compose.yml файдл, описывающий запуск описанных выше контейнеров.
